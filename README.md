# Senior Data Engineer Portfolio

Static portfolio site for Ahmed Hemad, built from the supplied CV and reference image direction.

## Files

- `index.html` - page content and sections
- `styles.css` - responsive visual design
- `script.js` - project filtering behavior
- `assets/ahmed-portrait.png` - generated portrait asset based on the supplied reference style
- `assets/ahmed-hemad-cv.pdf` - local CV file linked from the portfolio

## Customize

Update these placeholders before publishing:

- Display name if you prefer `Ahmed El Sayed` instead of `Ahmed Hemad`
- Any public phone number you want to add
- Project details, metrics, and client names that should be removed or anonymized

Open `index.html` directly in a browser to preview the site.
